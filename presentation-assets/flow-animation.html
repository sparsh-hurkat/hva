HOW IT WORKS

Six sources, one view of release risk

AZURE DEVOPS
Pulls every upcoming Release Work Item (RWI).
RWI COMMENTS
Reads repo name and branch from the RWI's comments.
PIPELINE RUN
Triggers a fresh build on that repo and branch.
MEND API
Called directly — flags vulnerable dependencies.
FORTIFY API
Called directly — flags static code vulnerabilities.
WIZ (PIPELINE)
Scans inside the pipeline run — read from there, not its own API call.
JFROG ARTIFACTORY
Flags Mend-detected packages published in the last 7 days.
SPARK AI
Scrapes the repo, then writes the summary and the suggested fix.
DASHBOARD
Release managers drill in, see the fix, and open the PR.
Six sources. One dashboard. Full visibility.
